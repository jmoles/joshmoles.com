# Josh Moles

Engineer and program manager whose work has spanned multiple computing
paradigms, from unconventional simulated chemistries to quantum. Currently
at IonQ, working on integrating conventional and quantum systems.
Previously at the U.S. Department of State, Google, Intel, and Northrop
Grumman. IEEE Senior Member, FAA Private Pilot, and amateur radio operator (NV2X).

## Projects

- [keccak-verilog](https://github.com/jmoles/keccak-verilog) — SystemVerilog implementation of SHA-3 / Keccak.
- [prompt-evolver](https://github.com/jmoles/prompt-evolver) — Evolve structured LLM agent prompts with multi-objective genetic algorithms, fully local via Ollama.
- [trail-runner](https://github.com/jmoles/trail-runner) — Evolutionary simulations on Santa Fe / John Muir trail problems, building on my [MS thesis](https://github.com/jmoles/ms-thesis).
- [petname.moles.dev](https://petname.moles.dev) — Static web port of Dustin Kirkland's petname utility for memorable, pronounceable names.

## Previously

- **Google** (2023–2025) — Technical Program Manager on Tensor G5 SoC (Pixel 10) and Android Auto connectivity.
- **U.S. Department of State** (2015–2022) — Lead Technology & Operations Program Manager for multimode communications systems supporting national security objectives.
- **Intel** (2011–2014) — Design Engineer on post-silicon validation for Intel Atom processors.
- **Northrop Grumman** (2009–2011) — Design Engineer on electronic warfare FPGA systems; deployed to Kandahar as field engineer on the DARPA [HALOE](https://www.darpa.mil/about/innovation-timeline/high-altitude-lidar) LIDAR program.
